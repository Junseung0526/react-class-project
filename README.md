# 뉴스 브리핑 대시보드

관심있는 주제의 최신 뉴스를 확인하고 스크랩할 수 있는 뉴스 대시보드 애플리케이션입니다.

## 주요 기능

- **맞춤 뉴스 검색**: 원하는 직무나 키워드를 입력하여 관련 뉴스를 필터링하고 검색할 수 있습니다.
- **뉴스 정렬**: 검색된 뉴스를 관련도순 또는 최신순으로 정렬하여 볼 수 있습니다.
- **실시간 뉴스**: 현재 가장 주목받는 실시간 뉴스를 확인할 수 있습니다.
- **분야별 토픽**: IT, 경제, 사회 등 주요 분야별 인기 뉴스를 모아볼 수 있습니다.
- **스크랩**: 관심 있는 뉴스는 '스크랩'하여 보관하고 언제든지 다시 볼 수 있습니다. (브라우저의 Local Storage에 저장됩니다.)
- **테마 전환**: 라이트 모드와 다크 모드를 모두 지원하여 눈이 편안한 환경에서 뉴스를 읽을 수 있습니다.

## 기술 스택

- **프론트엔드**: React (Vite)
- **스타일링**: CSS Modules
- **데이터 소스**: Naver 검색 API
- **아이콘**: Lucide React

## 시작하기

프로젝트를 로컬 환경에서 실행하는 방법은 다음과 같습니다.

### 1. 사전 준비

- [Node.js](https://nodejs.org/ko/) (LTS 버전 권장)가 설치되어 있어야 합니다.
- [네이버 개발자 센터](https://developers.naver.com/main/)에서 **검색** API 사용 신청 후 Client ID와 Client Secret을 발급받아야 합니다.

### 2. 환경 변수 설정

프로젝트 루트 디렉토리에 `.env` 파일을 생성하고 아래와 같이 발급받은 키를 입력합니다.

Vite 프로젝트에서는 환경 변수 이름 앞에 `VITE_` 접두사를 붙여야 합니다.

```
VITE_NAVER_CLIENT_ID="여기에 발급받은 Client ID를 입력하세요"
VITE_NAVER_CLIENT_SECRET="여기에 발급받은 Client Secret을 입력하세요"
```

**중요**: 이 프로젝트는 Vite의 프록시 기능을 사용하여 네이버 API의 CORS 정책을 우회합니다. `vite.config.js`에 관련 설정이 포함되어 있습니다.

### 3. 설치 및 실행

1.  프로젝트에 필요한 패키지를 설치합니다.
    ```bash
    npm install
    ```

2.  개발 서버를 실행합니다.
    ```bash
    npm run dev
    ```

3.  브라우저에서 `http://localhost:3000` 으로 접속하여 애플리케이션을 확인합니다.

## 📁 프로젝트 구조

주요 파일 및 디렉토리 구조는 다음과 같습니다.

```
/
├── public/              # 정적 에셋
├── src/
│   ├── assets/          # 폰트, 이미지 등
│   ├── components/      # 재사용 가능한 React 컴포넌트
│   ├── styles/          # CSS Module 파일
│   ├── utils/           # API 호출, 상수 등 유틸리티 함수
│   ├── App.jsx          # 메인 애플리케이션 컴포넌트
│   └── main.jsx         # 애플리케이션 진입점
├── .env.example         # 환경 변수 예시 파일 (실제 사용 시 .env로 복사)
├── .gitignore
├── index.html
├── package.json
└── vite.config.js       # Vite 설정 파일 (프록시 포함)
```
